# Workflow рекомендації

**1. Один tier за раз.** Зроби 4-5 листів Tier 1 у перший день. Подивися відгуки через 48 годин. Потім Tier 2. Потім Tier 3.

**2. Кастомізація — обов'язково.** У шаблонах є `[specific thing]` / `[Company is doing X]` — це не для краси. Лист без хоча б однієї конкретної фрази про компанію читається як спам. Мінімум — 30 секунд на їхньому сайті перед кожним листом.

**3. Час відправки.** Найкращі вікна для українського HR: вівторок-четвер, 10:00–11:30 або 14:00–16:00. Не понеділок ранок, не п'ятниця після обіду.

**4. Subject line — найважливіше.** Шаблони спеціально починаються з «Family Physician + Healthcare Software» а не з «Application» / «Hello» / «My CV». Унікальність subject = open rate.

**5. Який CV прикріплювати:**
- Українська компанія, лист українською → `CV_Eugene_UA.pdf`
- Українська компанія, лист англійською → `CV_Eugene_EN.pdf`
- Міжнародна компанія → `CV_Eugene_EN.pdf`
- Іноді доцільно прикріпити обидва, особливо для українських компаній — рекрутер може передати міжнародному hiring manager

**6. Tracking.** Google Sheet або Notion з колонками: Company / Channel / Contact / Date sent / Response / Status / Notes. Без структурованого трекінгу через 2 тижні забудеш кому що писав.

# Чого НЕ робити

- **Не згадуй бронювання, КВП, мобілізацію, ВЛК** у першому листі. Ніколи. Це одразу читається як «людина шукає бронь, а не роботу». Цю тему піднімаєш сам на фінальному етапі, коли вже є оффер на руках або майже-оффер.
- **Не вибачайся за бекграунд.** «Я знаю що не маю формального досвіду PM, але...» — викидай. Твій бекграунд це перевага, не дефект.
- **Не пиши довгих листів.** Якщо більше 8-10 рядків — ріж. Рекрутер скролить на телефоні, довгий лист = автоскіп.
- **Не використовуй «Шановні пані та панове».** Знайди ім'я. Якщо не вдається — «Доброго дня, команда [Company]».
- **Не згадуй зарплатні очікування** у першому листі. Це розмова для скринінгу.

# Що робити прямо зараз

1. **Налаштуй email signature** у Gmail (5 хвилин). Це one-time, далі автоматично під кожним листом.
2. **Створи tracking spreadsheet** — Google Sheets, простий: Company / Channel / Contact / Date sent / Response / Notes (10 хвилин).
3. **Відкрий цей файл** в окремій вкладці поряд з Gmail.
4. **Почни з однієї компанії.** Рекомендую Helsi.
   - Відкрий jobs.helsi.me або work.ua → знайди конкретну вакансію
   - Якщо є — використай шаблон 1A
   - Якщо немає — використай Universal Speculative
   - Знайди контакт HR (LinkedIn, сайт компанії, або підпис на вакансії)
   - Скопіюй шаблон, заміни [placeholders], додай 1-2 речення про конкретику компанії
   - Прикріпи CV (UA pdf якщо лист українською)
   - Відправ
5. **Запиши у tracking** — компанія, дата, ким є контакт.
6. **Перший лист найскладніший.** Після нього — звичка. Не сиди над 5 листами одночасно. Один за раз.

Коли відправиш перші 3-5 листів — повертайся, розкажи хто відповів, які питання задають. Доопрацюємо process по ходу.
